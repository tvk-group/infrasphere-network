## Replacing with your official files

Drop your final exported PNGs into this folder using the **same filenames**. The site picks them up automatically — no code changes needed.

Recommended export sizes:
- Full logos: ~440×144 px (2× for retina) or SVG
- Mark: 512×512 px minimum
- Favicon sizes: exact pixel dimensions as listed above
